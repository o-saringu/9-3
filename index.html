プログラム復習コンテンツ

return(値を返す)

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　float yourWeight=132;

　　 　float marsWeight=calculateMars(yourWeight);

　　 　println(marsWeight);　

　　 }

　　 　

　　 float calculateMars(float w){

　　 　float newWeight=w*0.38;

　　 　return newWeight;

　　 }

説明文

　これまで定義した関数はどれも、関数名の前にvoidというキーワードがついている。

　値を返す関数を作るときは、voidの代わりにその関数が返す値のデータ型を書く。

　関数の中には、返すデータを指定するreturn文がある。

　 returnが実行されるとその値をもとの関数に返す。

　　

　このプログラムでは、calculateMars()は火星表面における体重を計算し、その結果を返している。
